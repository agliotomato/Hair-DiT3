### 결과 분석

#### 1. FID (Fréchet Inception Distance) ↓

**무엇을 측정하나:** InceptionV3로 추출한 이미지 feature 분포를 실제 데이터셋 분포와 비교하는 지표. 두 분포 간의 Fréchet(Wasserstein-2) 거리로, **낮을수록 생성 이미지의 분포가 실제에 가깝다**. 단일 이미지 비교가 아니라 전체 샘플 집합의 분포 차이를 측정하므로 생성 모델의 전반적 품질 및 다양성을 종합적으로 반영한다. 색상·텍스처·형태 모두 포함.

- **FID hair:** Ours **85.75** vs SketchHairSalon 234.22 vs HairClipV2 287.28 → 약 2.7~3.4배 낮음. 헤어 영역 생성 품질이 실제 데이터 분포에 가장 근접한다.
- **FID boundary:** Ours **12.51** vs SketchHairSalon 54.54 vs HairClipV2 72.57 → 약 4~6배 낮음. 경계 처리 품질에서 압도적 우위.

#### 2. LPIPS (Learned Perceptual Image Patch Similarity) ↓

**무엇을 측정하나:** VGG 네트워크의 중간 feature map 간 거리로 인간의 시각적 인지와 유사한 지각적 유사도를 측정. 픽셀 단위 비교(PSNR, SSIM)와 달리 텍스처·스타일 차이에 민감하며, **낮을수록 지각적으로 GT에 가깝다**. 색상도 반영된다.

- **LPIPS_hair:** Ours **0.0256** vs SketchHairSalon 0.0499 vs HairClipV2 0.0757 → SketchHairSalon 대비 약 2배 낮음. 헤어 텍스처의 지각적 품질이 가장 우수하다.
- **Boundary LPIPS:** SketchHairSalon **0.0088** ≈ Ours 0.0097 vs HairClipV2 0.0467 → 두 모델이 근소한 차이로 경계 처리 품질이 유사. HairClipV2 대비 약 5배 낮다.

#### 3. SSIM (Structural Similarity Index) ↑

**무엇을 측정하나:** 밝기·대비·구조 세 요소를 분리해 측정하는 구조적 유사도 지표. **1에 가까울수록 GT와 구조적으로 유사**하다.

- **SSIM_bg:** SketchHairSalon **0.9127** > Ours 0.8608 > HairClipV2 0.6483. 배경 구조 보존은 SketchHairSalon이 우위. 배경을 거의 그대로 복사하는 방식의 특성상 당연한 결과.
- **SSIM_hair:** Ours **0.7450** > SketchHairSalon 0.6569 > HairClipV2 0.3426. 헤어 영역 구조 유사도는 Ours가 가장 높음. 새 SketchHairSalon 결과에서도 0.6569로 개선됐지만 Ours와는 0.088 차이.

#### 4. PSNR (Peak Signal-to-Noise Ratio) ↑

**무엇을 측정하나:** 픽셀 단위 MSE 기반 신호 대 잡음비(단위: dB). **높을수록 픽셀 수준에서 GT에 가깝다**. 텍스처 디테일보다 전체적인 밝기·색상 정확도에 민감하다.

- **PSNR_bg:** SketchHairSalon **47.34** >> Ours 30.88 > HairClipV2 24.06. Ours는 HairClipV2보다 약 6dB 높아 배경 보존 자체는 양호하나, SketchHairSalon 대비 약 16dB 차이는 배경 아티팩트 존재 가능성을 시사한다. Compositor ablation study 필요.

#### 5. ArcFace Cosine Similarity ↑

**무엇을 측정하나:** ArcFace 얼굴 인식 모델로 추출한 embedding 간 코사인 유사도. **1에 가까울수록 헤어 변경 후에도 동일 인물로 인식**됨을 의미한다.

- Ours **0.9817** vs SketchHairSalon 0.9942 vs HairClipV2 0.8098. Ours는 SketchHairSalon과 0.0125 차이로 매우 근접. HairClipV2 대비 약 0.17 높아 헤어 변경 시 얼굴 identity를 거의 완벽히 보존함을 확인.

#### 6. Edge IoU & Chamfer Distance

**무엇을 측정하나:**
- **Edge IoU:** Canny edge(생성 이미지) ∩ sketch edge / union. 높을수록 sketch 윤곽을 충실히 따름.
- **Chamfer Distance:** 생성 edge ↔ sketch edge 간 양방향 최소거리 평균(px). 낮을수록 sketch에 가깝다.

- **Edge IoU:** SketchHairSalon **0.0786** ≈ Ours 0.0778 >> HairClipV2 0.0551. 헤어 윤곽 충실도는 두 모델이 거의 동등.
- **Chamfer Dist:** Ours **6.01** ≈ SketchHairSalon 5.91 < HairClipV2 7.85. 이번 결과에서 SketchHairSalon이 근소하게 낮지만 실질적 차이는 없는 수준(0.1px).

→ **해석:** Ours는 sketch 형태 제어(Edge IoU, Chamfer)에서 SketchHairSalon과 대등하면서, 헤어 생성 품질(LPIPS_hair, FID, SSIM_hair)에서 압도적 우위. 경계 처리(Boundary LPIPS)는 SketchHairSalon과 거의 동등. 단, FID/LPIPS에 색상이 포함되어 있어 헤어 색상 재현 능력도 함께 우수한 것으로 해석된다.

#### 7. 약점: 배경 보존 (PSNR_bg / SSIM_bg) ⚠️

**PSNR_bg / SSIM_bg:** SketchHairSalon이 47.34 / 0.9127(거의 원본 유지)을 기록한 반면 Ours는 30.88 / 0.8608에 그침. HairClipV2(24.06 / 0.6483)보다는 우위이나, SketchHairSalon과의 격차가 여전히 크다.

**원인 추정:** 헤어 영역 외 배경까지 diffusion이 개입해 불필요한 수정이 발생하고 있을 가능성이 큼. latent compositor의 배경 보존 메커니즘 ablation study 필요.
